Project Name
VUT Digital Newsletter Website

Project Overview
The VUT Digital Newsletter Website is a web-based platform designed to present institutional news, announcements, and updates in a clear and accessible digital format. The project focuses on structured content presentation, responsive design, and usability for different devices.

Objectives
Digitally present newsletter content in an organized format
Improve accessibility of institutional information
Apply responsive web design principles
Demonstrate front-end web development skills

Features
Clean and structured newsletter layout
Responsive design for desktop and mobile devices
Navigation menu for different content sections
Organized presentation of articles and announcements

Technologies Used
HTML
CSS
JavaScript

Skills Demonstrated
Front-end web development
Responsive web design
Requirements interpretation
Information structuring and content layout

Project Type
Academic/ Personal Project
